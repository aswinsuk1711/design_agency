# Void Studio — Design Agency Homepage

A dark, editorial design agency landing page built with Next.js 14 and Tailwind CSS.

## Features

- Smooth scroll animations with Framer Motion
- Dark editorial aesthetic
- Responsive across mobile and desktop
- Project showcase and contact sections

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech

Next.js 14 · Tailwind CSS · TypeScript · Framer Motion
